# Contributing to claude-care-orchestrator

As the Healthcare DRI, I ensure that all contributions align with our 0→1 strategy.

## Cross-Functional Workflow

1. **Engineering:** All code must pass the `compliance_audit.py` gate.
2. **Policy/Legal:** Any changes to the `PROMPT_MASTER.md` must be reviewed for HIPAA-compliance.
3. **Research:** If you identify a failure in clinical reasoning, log it in `DEBUG_TRIAGE.md` for our Applied AI team.

## Our Culture

- **Low Ego, High Accountability:** We catch every ball.
- **Triage First:** Focus on the highest-value waste-reduction workflows first.
